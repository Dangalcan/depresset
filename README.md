## Instrucciones

Se debe elegir un conjunto de datos y analizarlo. Repositorio de datos:

* Kaggle: https://www.kaggle.com/
* UCI repository: https://archive.ics.uci.edu/

En análisis de datos tiene que ser reproducible, por ello el material a subir es:

1. Archivo .Rmd o R-notebook, junto con el .html que se genera.

2. Se debe realizar un pequeño resumen de las tareas realizadas al principio del R-notebook.

3. Carpeta data con los datos analizados.

Se debe tener en cuenta al exponer el análisis realizado:

* Problema que se plantea y los datos de trabajo.
* Análisis realizado.
* Conclusiones, dificultades, comentarios, etc.

En resumen, seguir la siguiente rúbrica: Rubrica.xlsx
